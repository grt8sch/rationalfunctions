Task 1

1/(x+a)

When the slider of variable a changes value, the vertical asymptote of the x value moves with it linearly. The variable a increasing causes shift left, and the variable a decreasing causes shift right.

The maximum and minimum number of asymptotes stays at one for a.

When 0<a<1 or a < 1, a still behaves normally.

1/(x^2+b)

When the slider of variable b increases value, the two vertical asymptotes get closer until they are greater than zero which causes the vertical asymptotes to not be real. When b decreases, the vertical asymptotes get farther apart from each other. The variable b increasing causes the asymptotes to compress inwards until they are imaginary, and the variable b decreasing causes the asymptotes to stretch away from each other.

When b < 0, there are two vertical asymptotes. When b = 0, there is one vertical asymptote. When b < 0, there are no vertical asymptotes.

When 0<b<1, the graph is like a hump that slowly gets smaller the greater value b is. When b is smaller than zero, there is no hump and two asymptotes.

x/(x+a)

The graph gets bigger when a is farther away from zero and causes the x value moves negatively to a.

The number of vertical and horizontal asymptotes stays at 1. This is the minimum and maximum asymptotes.

When each individual variable becomes between 0 and 1 or smaller than 1, each variable still behaves normally.

(x+a)/(x+b)

In the graph, when b gets closer to a, the graph gets smaller. When |b-a| in the graph increases, the vertical and horizontal stretch of the graph increase.

The number of vertical and horizontal asymptotes stays at 1. This is the minimum and maximum asymptotes.
